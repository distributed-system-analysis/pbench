Pbench

A benchmarking and performance

analysis framework


Three subsystems for big results.

Pbench Agent

The collection agent or harness, pbench-agent, is responsible for providing commands for running benchmarks across one or more systems, while properly collection the configuration of those systems, their logs, and specified telemetry from various tools (sar, vmstat, perf, etc).

The second sub-system included here is sever, or "back-ground tasks", which is responsible for achieving result tar balls, indexing them, and unpacking them for display.

Pbench is a harness that allows data collection from a variety of tools while running a benchmark. Pbench has some built-in script that run some common benchmarks, but the data collection can be run separately as well with a benchmark that is not built-in to pbench, or a pbench script can be written for the benchmark.
